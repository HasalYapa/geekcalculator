# **App Name**: Geek Calculator Hub

## Core Features:

- Matrix Calculator: Perform matrix addition, subtraction, and multiplication, calculate determinants and inverses (2x2, 3x3).
- Polynomial Solver: Solve quadratic and cubic equations with step-by-step solutions and discriminant display.
- Vector Calculator: Calculate vector addition, subtraction, dot product, cross product (3D), and angle between vectors.
- Unit Converter: Convert between physics units (length, mass, temperature, force, energy) using dropdown selections for instant conversion.
- Statistics Calculator: Calculate mean, median, mode, standard deviation, and variance from comma-separated input values.
- Base Converter: Convert between binary, octal, decimal, and hexadecimal bases with live conversion while typing.
- Save Calculations: Save recent calculations to local storage.

## Style Guidelines:

- Primary color: Deep purple (#6246EA) for a modern, sophisticated feel.
- Background color: Light gray (#F5F5F5) to ensure a clean and distraction-free workspace.
- Accent color: Teal (#2EC4B6) for interactive elements and highlights, providing a fresh contrast.
- Body and headline font: 'Inter', a sans-serif font known for its readability and modern aesthetic.
- Code font: 'Source Code Pro' for displaying code snippets, calculations and live conversions.
- Use crisp, technical icons appropriate for calculators, such as formula symbols or scientific symbols.
- Employ a clean and modern UI layout with rounded cards and shadow effects to emphasize depth and hierarchy.
- Add subtle transition animations on button clicks or when showing/hiding results to improve UX and provide visual feedback.
